📘 Caderno Temático — Cibersegurança
1. 🎯 Objetivo

Este projeto tem como objetivo estudar ataques comuns direcionados a usuários com alto privilégio em ambientes corporativos, utilizando o NotebookLM como ferramenta de apoio ao aprendizado.

O foco está no desenvolvimento de:

pensamento crítico
curadoria de fontes
engenharia de prompts
organização do conhecimento
2. 🧠 Tema

Usuários com alto nível de acesso (administradores, executivos e equipes técnicas) são alvos prioritários em ataques cibernéticos, pois possuem controle sobre sistemas críticos e dados sensíveis.

Este estudo analisa como esses ataques funcionam, quais vulnerabilidades são exploradas e como preveni-los.

4. ⚙️ Engenharia de Prompts

Pergunta:
"Explique ataques cibernéticos em empresas"

Problema:
Muito genérico → respostas superficiais

4.2 Prompt Intermediário

Pergunta:
"Quais são os ataques mais comuns contra usuários com privilégios elevados em empresas?"

Melhoria:
Maior foco no público-alvo

4.3 Prompt Avançado

Pergunta:
"Explique os principais ataques direcionados a usuários privilegiados, incluindo exemplos reais, etapas do ataque, vulnerabilidades exploradas e medidas de prevenção"

Resultado:
Resposta mais estruturada e aplicável

4.4 Prompt Crítico

5. 📖 Resumo Estruturado
5.1 Phishing Direcionado (Spear Phishing)
Ataques personalizados
Foco em executivos
Uso de engenharia social
5.2 Credential Dumping
Extração de credenciais
Comprometimento de contas administrativas
5.3 Privilege Escalation
Elevação indevida de acesso
Exploração de falhas no sistema
5.4 Engenharia Social
Manipulação psicológica
Exploração de confiança
5.5 Insider Threat
Uso indevido de acesso interno
6. 📘 Glossário
Phishing: tentativa de enganar usuários para roubar dados
Spear Phishing: phishing direcionado
Privilege Escalation: aumento indevido de privilégios
Credential Dumping: roubo de credenciais
Engenharia Social: manipulação psicológica
7. 🧠 Miniguia de Estudo
7.1 Resumo Geral

Usuários privilegiados são os principais alvos de ataques, pois concentram acesso crítico dentro das organizações.

7.2 Conceitos-chave
Alto privilégio implica alto risco
Ataques combinam falhas humanas e técnicas
Segurança depende de múltiplas camadas
7.3 Prompts Reutilizáveis
